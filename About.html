About The Game

Our game Puzzle Seeing is a multiplayer game where you can learn fun facts about famous landmarks by solving funny puzzles. It uses html, css, javascript, JQuery, and Google Street View API. You're spawned on the spot near the famous place, and you have to be able to find the famous landmarks by the solving the puzzles the game provided for you. You have to use the arrows on Google Street View to find the exact destinations. The more puzzles, the more points you get. You can compete with friends and families to find out who has better knowledge about the world.
